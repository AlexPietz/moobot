# moobot

now actually won't crash every 0.00001 seconds

## prereqs

* python 3.4 or 3.5
* virtualenv
* pip

## install

**set up virtualenv**

`virtualenv -p $(which python3) .venv`

**install dependencies**

`pip install -r requirements.txt`

**next, add your discord token in the moobot.py file**

**then, run**

`python moobot.py`
